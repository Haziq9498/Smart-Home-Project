# Smart Security System with Environmental Monitoring

A comprehensive smart security and monitoring system using ESP32 and ESP RainMaker. This project provides intelligent access control, environmental monitoring, and automated security responses. You can control and monitor it via voice assistants or mobile app.

## Features

- Keypad-based access control with PIN authentication
- Automatic security alerts on multiple failed attempts
- Temperature and humidity monitoring with environmental awareness
- Voice assistant integration (Alexa / Google Assistant)
- OTA firmware updates for seamless maintenance
- Multi-level alert system with visual and cloud notifications
- Smart lighting integration with automated responses
- Built with FreeRTOS for reliable multitasking
- Real-time security status via ESP RainMaker cloud

## Components Used

- ESP32-WROOM-32 Development Board
- 4x4 Matrix Keypad for secure access control
- DHT11 Sensor for temperature and humidity monitoring
- LED Indicators for system status and alarms
